# Smartphone Store

A simple smartphone store website project deployed on Vercel for learning frontend development and web deployment workflows.

---

## Features

- Smartphone product showcase
- Responsive web interface
- Product browsing layout
- Basic website navigation
- Vercel deployment

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Vercel

---
